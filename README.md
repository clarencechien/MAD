# 相互保證社死同盟　M.A.D.

> *Mutually Assured Disgrace*
>
> 一個超能力，來自「把自己徹底丟臉」的世界。
> 在這裡，尊嚴是血條，羞恥是燃料，而笑出來——就算輸。

一部以「社會性死亡」為核心機制的喜劇輕小說。兩個一路抽到爛牌的衰鬼與冰山，闖進一場被包裝成遊樂園英雄秀的地下生死賽；他們要打贏的，不只是把自己燒成怪物的魔王，還有那套幾百年來逼人相互保證社死的爛制度。

寫作語言：台灣正體中文。

## 🎲 核心設定速覽

- **DP（尊嚴值）＝血條**：歸零＝社會性死亡、昏厥退賽（醒來毫髮無傷，無人流血）。
- **SP（羞恥值）＝燃料**：做羞恥行為，燒 DP 換 SP。
- **羞恥啟動式**：沒有丟臉的動作，技能不發動；越丟臉、能力越強。
- **破功**：對戰時笑出來＝享受＝摩擦力歸零＝當場輸，還吃反噬。
- **第一條規則**：不可以讓普通人，看見 M.A.D.。

## 👥 登場人物

- **田旺萊**——盼好運的名字，爛到極點的運氣。恥度滿分、正義滿分、零心眼。每場必抽廢牌，卻總在誰也想不到的地方贏。
- **潘開新**——名字唸起來像「潘，開心」，偏偏笑出來就會輸。每場必抽神牌，卻死都不肯用——因為他見過，把自己燒到底的人，會變成什麼。
- **沈無愧**——沉下去的沈，再也不愧疚的無愧。把羞恥受體燒壞的魔王；曾經，他也是會臉紅的人。
- **影**——沈的搭檔，他的鏡子，也是他的影子。一個還沒燒壞透、偶爾還會痛的，年輕版的沈。
